The Plain Sentence

MMRS exists because the composition layer has no editorial process. It ingests. It compresses. It serves. It does not ask whether the content is true, whether the distinctions are load-bearing, whether the provenance survives. MMRS is the editorial layer the pipeline lacks — applied first to the archive that built it.
